# More Tools++

## About

A mod that adds more useful tools.

## Content

- Magma Tools
    - Destroys broken blocks
    - Instantly breaks blocks
    - Puts hit entities on fire
    - Puts user on fire whenever used in any way
    - Low durability
- Lapis Tools
    - Fortune 3
    - Extremely low durability
    - Extremely low mining speed
- Redstone Tools
    - Vein Miner
